# 有向非巡回グラフを用いた年表生成システムの開発
本プロジェクトでは、有向非巡回グラフ（DAG）を活用して、テキストから出来事の順序・因果関係を解析し、高精度な年表を生成するシステムの開発を目指します。
This project aims to develop a system that generates a highly accurate timeline by utilizing Directed Acyclic Graphs (DAGs) to analyze the sequence and causal relationships of events from text.
